Nağme — Privacy Policy

Last updated: April 2, 2026

1. Overview

Nağme is a free, ad-free chromatic tuner application. User privacy is our top priority.

2. Data Collection

Nağme does not collect, store, or share any personal data with third parties.

  • No account creation required
  • No analytics or tracking code
  • No advertising SDKs
  • No internet connection required

3. Microphone Usage

Nağme requests access to the device microphone for tuning functionality.

  • Microphone data is used only for real-time frequency analysis
  • Audio data is processed on-device and is never recorded, stored, or transmitted
  • Microphone access is only used while the user is actively tuning
  • The application does not access the microphone in the background

4. Data Storage

Application settings (such as reference frequency and selected instrument) are stored only in the device's local storage (SharedPreferences). This data never leaves the device.

5. Children's Privacy

Nağme is suitable for all age groups and does not collect any data from children.

6. Third-Party Services

Nağme does not use any third-party services, analytics tools, or advertising networks.

7. Changes

This policy may be updated. Changes will be published on this page.
